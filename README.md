<h2>Overview</h2>
<p>
A full-stack Todo application built with <strong>Node.js</strong>, <strong>Express</strong>, and vanilla 
<strong>HTML, CSS, and JavaScript</strong>.  
The project demonstrates RESTful API design, proper HTTP status code handling, and dynamic frontend rendering.
</p>

<h2>Key Features</h2>
<ul>
  <li>Add, update, and delete tasks</li>
  <li>Auto-generated unique task IDs</li>
  <li>Priority system (High / Medium / Low)</li>
  <li>Colour-coded priority tags</li>
  <li>In-memory data handling</li>
  <li>Clean frontend/backend separation</li>
</ul>

<h2>Technical Highlights</h2>
<ul>
  <li>RESTful API routes (GET, POST, PUT, DELETE)</li>
  <li>Proper HTTP status codes (200, 201, 400, 404, 500)</li>
  <li>Express middleware & JSON parsing</li>
  <li>Dynamic DOM manipulation</li>
  <li>State-based UI interaction</li>
</ul>
